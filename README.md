### 📩 Classification de SMS Spam *(NLP & Deep Learning)*
**Prétraitement Textuel & Apprentissage Profond**

📌 **Objectif**  
Développer un système de classification automatique capable de distinguer les messages SMS légitimes (*Ham*) des messages indésirables (*Spam*).

📊 **Dataset**
- SMS Spam Collection (UCI Machine Learning Repository)
- 5 572 messages
- Dataset déséquilibré (≈ 86.6 % Ham / 13.4 % Spam)

🛠️ **Méthodologie**
- Nettoyage et normalisation du texte
- Tokenisation et lemmatisation
- Vectorisation TF-IDF
- Réduction de dimension (Autoencodeur)
- Modèle avancé basé sur **BERT**
- Évaluation avec Accuracy, Precision, Recall et AUC-ROC

📈 **Résultats**
- **TF-IDF** : très bon compromis entre performance et simplicité  
- **BERT** : meilleure performance globale grâce à la compréhension sémantique  
- **Autoencodeur** : échec sur la classe minoritaire (recall spam = 0 %)

📚 **Leçon clé**  
> L'accuracy seule est trompeuse sur un dataset déséquilibré.  
> Les métriques comme le Recall et l'AUC-ROC sont essentielles.
